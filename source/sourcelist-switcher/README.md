## Dependencies

- apt-transport-https
- bc
  
```bash
$ apt install wget apt-transport-https bc
```

## Installation

```bash
$ sudo bash -c "$(wget -q -O - https://raw.githubusercontent.com/Vincent0700/static-libraries/master/source/sourcelist-switcher/run.sh)"
```
