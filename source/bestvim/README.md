## Dependencies

- wget
- apt-transport-https
- git
- vim
  
```bash
$ sudo apt install wget apt-transport-https git vim
```

## Installation

```bash
$ sudo bash -c "$(wget -q -O - https://raw.githubusercontent.com/Vincent0700/static-libraries/master/source/bestvim/run.sh)"
```
